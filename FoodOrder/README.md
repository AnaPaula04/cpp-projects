# Food Order Self-Service System

A C++ console application for a self-service food ordering system.

## Features

- Order pizza, drinks, chips, and salad
- Automatic price calculation
- Receipt generation with change calculation

## How to Compile and Run

### On Mac/Linux:

```bash
g++ FoodOrder.cpp -o FoodOrder
./FoodOrder
```
